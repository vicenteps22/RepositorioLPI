# Multiplicação de Arrays 2d Simulada em Arrays 1d

Esse código faz a multiplicação de arrays 2d simuladas em arrays 1d.

---

## 📜 Descrição

Este programa implementa a multiplicação de duas matrizes utilizando arrays unidimensionais para simular matrizes bidimensionais. A ideia principal é armazenar as matrizes como vetores e calcular as posições dos elementos manualmente, o que pode ser útil para otimizações ou limitações de memória.

---

## 🛠️ Tecnologias Utilizadas

- 🧠 **Linguagem:** C
- 🖥️ **IDE:** VSCode (opcional)
- 🔧 **Compilador:** GCC

---

## 🚀 Como Executar

### ✔️ Pré-requisitos

Possuir um compilador C instalado:

- **Linux:**  
GCC costuma vir pré-instalado, mas caso não, execute:  
```bash
sudo apt update
sudo apt install build-essential
```

- **Windows:**  
Instale o [MinGW-w64](https://www.mingw-w64.org/) e configure as variáveis de ambiente para usar o GCC no terminal.

---

### ✔️ Compilar e Executar

1️⃣ Salve o código em um arquivo com a extensão .c.  
Exemplo: ```arquivo.c```

2️⃣ Abra o terminal e navegue até o diretório onde está o arquivo:  
```bash
cd caminho/para/sua/pasta
```

3️⃣ Compile o programa:  
```bash
gcc arquivo.c -o Programa
```

4️⃣ Execute o programa:  
```bash
./Programa
```
