# Exemplos Arrays

Este é um programa usado para demonstrar conceitos básicos de manipulação de arrays em C.

---

## 📜 Descrição

Este programa em C realiza duas operações principais:

- Cálculo de frequência e histograma:
Conta a frequência de respostas em um array e exibe um histograma visual usando asteriscos (*).

- Soma dos elementos de um array:
Soma os valores de outro array e imprime cada elemento com a soma total.

---

## 🛠️ Tecnologias Utilizadas

- 🧠 **Linguagem:** C
- 🖥️ **IDE:** VSCode (opcional)
- 🔧 **Compilador:** GCC

---

## 🚀 Como Executar

### ✔️ Pré-requisitos

Possuir um compilador C instalado:

- **Linux:**  
GCC costuma vir pré-instalado, mas caso não, execute:  
```bash
sudo apt update
sudo apt install build-essential
```

- **Windows:**  
Instale o [MinGW-w64](https://www.mingw-w64.org/) e configure as variáveis de ambiente para usar o GCC no terminal.

---

### ✔️ Compilar e Executar

1️⃣ Salve o código em um arquivo com a extensão .c.  
Exemplo: ```arquivo.c```

2️⃣ Abra o terminal e navegue até o diretório onde está o arquivo:  
```bash
cd caminho/para/sua/pasta
```

3️⃣ Compile o programa:  
```bash
gcc arquivo.c -o Programa
```

4️⃣ Execute o programa:  
```bash
./Programa
```
